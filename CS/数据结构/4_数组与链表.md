### 数组的优点与局限性
- 优点
  - 控件效率高
  - 支持随机访问
  - 缓存局部性
- 局限性
  - 插入与删除效率低
  - 长度不可变
  - 控件浪费
### 数组典型应用
- 随机访问
- 排序和搜索
- 查找表
- 机器学习
- 数据结构实现

### 数组 和 链表
|  | 数组 | 链表 |
| ------ | ------ | ------ |
| 存储方式 | 连续内存空间 | 分散内存空间 |
| 容量扩展 | 长度不可变 | 长度可变 |
| 内存效率 | 元素占用内存少但可能浪费空间 | 元素占用内存多 |
| 访问元素 | O(1) | O(n) |
| 添加元素 | O(n) | O(1) |
| 删除元素 | O(n) | O(1) |
